gitbash 的使用

其实也不算怎么使用，就是零星的把今天用到的东西记下来
gitbash里面很多命令和linux一样，甚至还有vim编辑器

  • 常用命令
    • pwd查看当前路径
    • mkdir xxxx新建文件夹
    • touch xxx.xx新建文件
    • rm xxx.xx删除文件
    • rm -r xxx删除文件夹
    • ls查看当前文件夹项目
    • vim xxx.xx编辑文件
  • vim 的使用
    • a或者i进入编辑模式
    • esc返回到命令模式
    • :wq保存退出
    • :q!强制退出不保存
    • ZZ退出 vim
      以后再补充
      使用 hjkl 移动光标貌似做不到

git 的使用

  • 本地仓库
    • git init 仓库初始化
    • git status 仓库状态
    • git add . 添加到暂存区
    • git commit -m "一些描述" 提交到本地库
    • git log本地库日志 回滚后看不到更晚的版本
    • git reflog本地库操作日志 可以看到所有版本 hash 及 HEAD 位置及曾经的操作
    • git reset --hard xxxx回滚到 xxxx 版本
      命令很多，用到的以后再补充
